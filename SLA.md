# PackArcade2 Service Level Agreement (SLA)

**Last Updated: March 18, 2026**

## 1. 🎯 Overview

PackArcade2 is a free, community-driven platform. We do our best to keep things running, but we're human! This SLA outlines what you can expect from our service.

## 2. ⏰ Server Availability

Our server typically runs during these hours:

| Time Period | Status |
|------------|--------|
| 6:00 AM - 3:00 PM | ✅ Online |
| 3:00 PM - 7:00 PM | ⚠️ Maintenance Window |
| 7:00 PM - 9:00 PM | ✅ Online |
| 9:00 PM - 6:00 AM | 🌙 Offline (resting) |

*Times are approximate and may vary based on real-life schedules*

## 3. 📊 Performance Expectations

| Metric | Expected Performance |
|--------|---------------------|
| Page Load | < 2-3 seconds |
| API Response | < 500ms |
| Build Time | < 1-2 minutes |
| File Upload | < 50MB per file |
| Uptime (target) | 97% during online hours |

## 4. 🔧 Maintenance

- Scheduled maintenance occurs during the 3PM-7PM window
- We'll announce major maintenance 24 hours in advance when possible
- Emergency maintenance may happen without notice

## 5. 💾 Data Backup

- User projects are backed up weekly
- Backups stored on offline media
- We recommend users keep local copies of important projects

## 6. 🆓 Support Commitment

| Issue Type | Response Time |
|------------|---------------|
| Critical bugs | < 24 hours |
| Feature requests | < 72 hours |
| General questions | < 48 hours |
| Spam | Whenever we get to it |

## 7. 🚫 Limitations

- Maximum file size: 50MB
- Maximum projects per user: Unlimited (for now!)
- Maximum backend processes: 1 per project
- No guaranteed 100% uptime (we're free!)

## 8. 💝 Our Promise

We promise to:
- Do our absolute best to keep the server running
- Be transparent about issues
- Never charge for core features
- Listen to community feedback
- Have fun building cool stuff!
